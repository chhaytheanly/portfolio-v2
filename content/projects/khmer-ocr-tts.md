# Khmer Text-to-Speech & OCR System

An end-to-end Khmer OCR extraction and Text-to-Speech pipeline using transfer learning and advanced image preprocessing techniques.

## Project Overview

This project addresses the challenge of Khmer text recognition and speech synthesis under limited-resource conditions. By leveraging transfer learning and advanced preprocessing, the system achieves robust performance on real-world documents.

## Key Features

- **Khmer OCR**: Advanced optical character recognition optimized for Khmer script
- **Text-to-Speech**: Natural speech synthesis for Khmer language
- **Image Preprocessing**: Denoising, binarization, and geometric correction for enhanced OCR accuracy
- **Transfer Learning**: Adapted pre-trained vision-language and speech models for Khmer
- **CUDA Optimization**: GPU-accelerated training and inference for reduced latency
- **RESTful API**: FastAPI-based scalable model serving

## Technical Stack

- **Backend**: FastAPI for RESTful API development
- **ML Framework**: PyTorch with CUDA acceleration
- **Models**: HuggingFace Transformers for vision-language and speech models
- **Image Processing**: OpenCV for preprocessing pipeline
- **Deployment**: Optimized for production serving with low latency

## Achievements

- Successfully adapted large language models for low-resource Khmer language
- Achieved high accuracy on real-world document images
- Built scalable API for integration with downstream applications
